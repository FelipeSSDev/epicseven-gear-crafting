## Setup - Requirements

1. Create a folder called `src`
2. put the JSON there;
    - `dados.json` is the expected name;

3. Download Node v16 or newer;
4. Run `npm run build` to build the JavaScript files;
5. Run the JavaScript files with `npm run dev`;
